# RamsBrewOrderSystem

Campus Coffee Shop Order Builder (CSC325 Lab)

## Design Patterns Used
- Factory Pattern: Creates different beverage types (Coffee, Latte, Cappuccino, Tea)
- Builder Pattern: Builds customized beverages step by step

## Menu
- Coffee (Hot/Iced)
- Latte (Hot/Iced)
- Cappuccino (Hot/Iced)
- Tea (Hot/Iced)

## Customizations
- Size: Small, Medium, Large
- Milk: Whole, Skim, Almond, Oat, Soy
- Shots: 1, 2, 3 (espresso drinks only)
- Sweetener: Sugar, Honey, Stevia, None
